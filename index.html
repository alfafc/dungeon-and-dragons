Reglas Básicas

Los Dados

d20 - El dado principal. Se usa para ataques, salvaciones y pruebas de habilidad.

d4, d6, d8, d10, d12 - Dados de dano. Cada arma usa un dado diferente.

d100 - Para eventos aleatorios y tablas especiales.

Combate

Iniciativa: Cada combatiente tira 1d20 + modificador de Destreza. El mayor va primero.

Atacar: Tira 1d20 + modificador de ataque. Si iguala o supera la Clase de Armadura (CA) del enemigo, le pegas.

Dano: Tira el dado de dano de tu arma + modificador de Fuerza (o Destreza para armas a distancia).

Critico: Si sacas 20 natural en el d20, es critico: tiras el doble de dados de dano.

Pifia: Si sacas 1 natural, fallas automaticamente.

Estadísticas

FUE (Fuerza) - Dano cuerpo a cuerpo, cargar cosas.

DES (Destreza) - Esquivar, sigilo, ataques a distancia.

CON (Constitucion) - Puntos de vida, resistencia.

INT (Inteligencia) - Magia arcana, conocimiento.

SAB (Sabiduria) - Percepcion, magia divina.

CAR (Carisma) - Persuasion, intimidacion, liderazgo.

Modificadores

Cada estadistica tiene un modificador: (valor - 10) / 2, redondeado hacia abajo.

Ej: Fuerza 16 = modificador +3, Fuerza 8 = modificador -1
